# FSTtech E-Commerce Website

Welcome to the FSTtech E-Commerce Website! This project aims to provide an online platform for customers to explore and purchase a wide range of products offered by FSTtech. The website is developed using PHP, JS, HTML5, CSS, Bootstrap, and MySQL technologies.

## Objectives

The main objectives of this website are:
1. Allow users to discover FSTtech's product offerings online.
2. Provide a seamless e-commerce experience for customers to place orders.

## Specific Features

The website includes the following key features:
1. Product Catalog: A user-friendly interface displaying detailed information for each product.
2. Customer Account Management: Secure login and a shopping cart for each customer.
3. Secure Payments: Online payment options, including credit card, cash on delivery, and PayPal.

## Additional Features

In addition to the core features, the website offers the following functionalities:
1. Promotions Management: Personalized promotions based on customer preferences and flash sales for limited time offers.

## Technologies Used

The website is developed using the following technologies:
- Front-End: HTML5, CSS, Bootstrap, JavaScript (JS).
- Back-End: PHP for server-side functionalities.
- Database: MySQL for data storage.

## Installation

To run the FSTtech E-Commerce Website locally on your machine, follow these steps:

1. Clone the repository: `git clone https://github.com/FSTtech/e-commerce.git`
2. Set up a local server (e.g., XAMPP, WAMP) and import the MySQL database from the `database` folder.
3. Launch the website from your web server.
